# 调试工具

## 📋 目录说明

本目录用于存放调试工具（GDB、LLDB、ADB）的学习文档、使用指南和实践案例。

## 📚 学习内容

- GDB 调试 Native 代码
- LLDB 调试技巧
- ADB 命令使用
- 断点、单步、变量查看
- core dump 分析
- 多线程调试

## 📝 学习产出

- GDB 使用指南
- LLDB 使用指南
- ADB 命令参考
- 调试技巧总结

## 🔗 工具资源

- [GDB 官方文档](https://www.gnu.org/software/gdb/documentation/)
- [LLDB 官方文档](https://lldb.llvm.org/)
- [ADB 官方文档](https://developer.android.com/studio/command-line/adb)

## 🔍 实践内容

- 使用 GDB 调试 Native 崩溃
- 使用 LLDB 调试 Android 应用
- 使用 ADB 收集日志和文件
