# 性能追踪工具

## 📋 目录说明

本目录用于存放性能追踪工具（Systrace、Perfetto、ftrace）的学习文档、使用指南和实践案例。

## 📚 学习内容

- Perfetto 使用
- Systrace 分析
- ftrace 使用
- 性能问题定位技巧
- 自定义 trace 点

## 📝 学习产出

- Perfetto 使用指南
- Systrace 分析指南
- ftrace 使用指南
- 性能分析案例

## 🔗 工具资源

- [Perfetto 官方文档](https://perfetto.dev/)
- [Systrace 官方文档](https://developer.android.com/topic/performance/tracing)
- [ftrace 文档](https://www.kernel.org/doc/Documentation/trace/ftrace.txt)

## 🔍 实践内容

- 使用 Perfetto 分析 ANR 问题
- 使用 Systrace 分析性能瓶颈
- 使用 ftrace 跟踪 Kernel 事件
