# ANR 分析工具

## 📋 目录说明

本目录用于存放 ANR 分析工具的使用指南、分析脚本和案例。

## 📚 工具学习

1. **traces.txt 分析**
   - traces.txt 文件格式
   - 堆栈信息提取
   - 线程状态分析

2. **systrace/perfetto 分析**
   - 时间线分析
   - 主线程阻塞点定位
   - CPU 使用率分析

3. **logcat 分析**
   - ANR 日志提取
   - 关键时间点定位

4. **自动化分析脚本**
   - traces.txt 自动解析
   - ANR 关键信息提取

## 📝 学习产出

- 工具使用指南
- 自动化分析脚本
- 分析案例报告

**提示**：熟练掌握工具使用是快速定位 ANR 问题的关键。
