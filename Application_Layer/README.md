# 应用层 (Application Layer)

对应安卓架构图最顶层：Android Apps、Privileged Apps、Device Manufacturer Apps、Android API。

## 子目录与知识点

- **Android_Apps**：系统应用、PKMS（安装/升级/应用异常）
- **Privileged_Apps**：特权应用相关
- **Device_Manufacturer_Apps**：设备制造商应用
- **Android_API**：应用通过 API 与系统交互

本层内容由 ACKLearning 与 stabilityLearning 融合后按主题归类。
