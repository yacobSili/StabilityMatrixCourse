## 1. 主流厂商分区对比

### 1.1 Google Pixel

- 使用动态分区
- 标准A/B分区
- 无独立recovery分区

### 1.2 Samsung

- 可能有Knox相关分区
- 可能有OEM特定分区
- 分区命名可能不同

### 1.3 Xiaomi

- 可能有MIUI特定分区
- 可能有cust分区

## 2. 厂商特定分区

不同厂商可能有额外的分区：
- OEM分区
- 厂商特定数据分区
- 安全相关分区

---

## 3. 分区命名差异

某些厂商可能使用不同的分区命名：
- 使用平台路径而非by-name
- 使用厂商特定命名

---

## 总结

厂商差异要点：

1. **标准分区**：所有厂商都有
2. **额外分区**：厂商可能添加
3. **命名差异**：可能不同

**查看设备分区**：
```bash
adb shell ls -l /dev/block/by-name/
```
