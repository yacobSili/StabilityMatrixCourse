# Java Exception 基础理论

## 📋 目录说明

本目录用于存放 Java Exception 相关的基础理论知识、JVM 异常机制和 OOM 原理。

## 📚 学习内容

1. **Java 异常处理机制**
   - 异常类型（Checked、Unchecked）
   - 异常传播机制
   - JVM 异常表（exception table）

2. **OOM 原理**
   - Java 堆内存管理
   - GC 机制（标记清除、复制、标记整理）
   - GC 算法（Serial、Parallel、CMS、G1、ZGC）
   - OOM 触发条件
   - OOM 类型（堆、元空间、直接内存）

## 📝 学习产出

- JVM 异常机制分析文档
- OOM 原理分析文档
- GC 机制分析文档

## 🔗 关键概念

- **OutOfMemoryError**: 内存不足异常
- **StackOverflowError**: 栈溢出异常
- **ClassNotFoundException**: 类未找到异常

---

**提示**：理解 JVM 机制是分析 Java Exception 的基础。
