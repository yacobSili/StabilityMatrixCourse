# 窗口添加与布局流程

## 引言

窗口的添加和布局是 WMS 的核心功能。当应用调用 `WindowManager.addView()` 时，会触发一系列复杂的流程，包括窗口添加、布局计算、Surface 创建等。本文将详细分析这些流程。

---

### 2.2 WMS.addWindow 完整流程

```java
// WindowManagerService.java
public int addWindow(Session session, IWindow client, LayoutParams attrs,
        int viewVisibility, int displayId, int requestUserId, ...) {
    
    int[] appOp = new int[1];
    final int type = attrs.type;
    
    synchronized (mGlobalLock) {
        // ======== 阶段 1: 权限检查 ========
        int res = mPolicy.checkAddPermission(type, isRoundedCornerOverlay, attrs.packageName,
                appOp);
        if (res != ADD_OKAY) {
            return res;
        }
        
        // ======== 阶段 2: 获取显示内容 ========
        final DisplayContent displayContent = getDisplayContentOrCreate(displayId, attrs.token);
        if (displayContent == null) {
            return WindowManagerGlobal.ADD_INVALID_DISPLAY;
        }
        
        // ======== 阶段 3: 获取/创建 WindowToken ========
        WindowToken token = displayContent.getWindowToken(attrs.token);
        
        // Token 检查
        final int rootType = getRootType(attrs);
        if (token == null) {
            if (rootType >= FIRST_APPLICATION_WINDOW && rootType <= LAST_APPLICATION_WINDOW) {
                // 应用窗口必须有有效的 Activity token
                return WindowManagerGlobal.ADD_BAD_APP_TOKEN;
            }
            if (rootType == TYPE_INPUT_METHOD) {
                // IME 窗口必须有有效的 token
                return WindowManagerGlobal.ADD_BAD_APP_TOKEN;
            }
            // 其他类型创建新 token
            token = new WindowToken.Builder(this, attrs.token, type)
                    .setDisplayContent(displayContent)
                    .setOwnerCanManageAppToken(session.mCanAddInternalSystemWindow)
                    .build();
        }
        
        // ======== 阶段 4: 创建 WindowState ========
        final WindowState win = new WindowState(this, session, client, token,
                parentWindow, appOp[0], attrs, viewVisibility, session.mUid,
                userId, session.mCanAddInternalSystemWindow);
        
        // 检查客户端是否存活
        if (win.mDeathRecipient == null) {
            return WindowManagerGlobal.ADD_APP_EXITING;
        }
        
        // ======== 阶段 5: 策略验证 ========
        final DisplayPolicy displayPolicy = displayContent.getDisplayPolicy();
        res = displayPolicy.validateAddingWindowLw(attrs, callingPid, callingUid);
        if (res != ADD_OKAY) {
            return res;
        }
        
        // ======== 阶段 6: 打开输入通道 ========
        final boolean openInputChannels = (outInputChannel != null);
        if (openInputChannels) {
            win.openInputChannel(outInputChannel);
        }
        
        // ======== 阶段 7: 关联窗口 ========
        win.attach();
        mWindowMap.put(client.asBinder(), win);
        win.mToken.addWindow(win);
        
        // ======== 阶段 8: 配置 Insets ========
        displayContent.getInsetsStateController().onWindowAdded(win);
        
        // ======== 阶段 9: 调整窗口序列 ========
        win.mToken.addWindow(win);
        displayPolicy.addWindowLw(win, attrs);
        
        // ======== 阶段 10: 更新焦点 ========
        boolean imMayMove = true;
        if (type == TYPE_INPUT_METHOD) {
            displayContent.setInputMethodWindowLocked(win);
            imMayMove = false;
        }
        
        if (win.canReceiveKeys()) {
            focusChanged = updateFocusedWindowLocked(UPDATE_FOCUS_WILL_ASSIGN_LAYERS,
                    false /*updateInputWindows*/);
        }
        
        // ======== 阶段 11: 分配层级 ========
        displayContent.assignWindowLayers(false /* setLayoutNeeded */);
        
        return res;
    }
}
```

### 2.3 WindowState 创建

```java
// WindowState 构造函数
WindowState(WindowManagerService service, Session s, IWindow c, WindowToken token,
        WindowState parentWindow, int appOp, LayoutParams a, int viewVisibility,
        int ownerId, int showUserId, boolean ownerCanAddInternalSystemWindow) {
    
    mSession = s;
    mClient = c;
    mToken = token;
    mAttrs.copyFrom(a);
    mViewVisibility = viewVisibility;
    mPolicy = mWmService.mPolicy;
    
    // 计算层级
    if (mAttrs.type >= FIRST_SUB_WINDOW && mAttrs.type <= LAST_SUB_WINDOW) {
        // 子窗口
        mSubLayer = mPolicy.getSubWindowLayerFromTypeLw(a.type);
        mIsChildWindow = true;
    } else {
        mSubLayer = 0;
        mIsChildWindow = false;
    }
    
    // 创建 InputWindowHandle
    mInputWindowHandle = new InputWindowHandle(
            mToken.getInputApplicationHandle(mDisplayContent), displayId);
    
    // 创建动画控制器
    mWinAnimator = new WindowStateAnimator(this);
}
```

## 3. 布局计算流程

### 3.1 relayoutWindow 概述

```java
// WindowManagerService.java
public int relayoutWindow(Session session, IWindow client, LayoutParams attrs,
        int requestedWidth, int requestedHeight, int viewVisibility, int flags,
        int seq, int lastSyncSeqId, ClientWindowFrames outFrames,
        MergedConfiguration mergedConfiguration, SurfaceControl outSurfaceControl,
        InsetsState outInsetsState, InsetsSourceControl[] outActiveControls,
        Bundle outSyncIdBundle) {
    
    int result = 0;
    synchronized (mGlobalLock) {
        final WindowState win = windowForClientLocked(session, client, false);
        
        // ======== 阶段 1: 更新请求的大小 ========
        win.setRequestedSize(requestedWidth, requestedHeight);
        
        // ======== 阶段 2: 更新可见性 ========
        final boolean viewVisibilityChanged = ...;
        if (viewVisibilityChanged) {
            win.mViewVisibility = viewVisibility;
        }
        
        // ======== 阶段 3: 更新窗口属性 ========
        if (attrs != null) {
            mPolicy.adjustWindowParamsLw(win, attrs);
            win.mAttrs.copyFrom(attrs);
        }
        
        // ======== 阶段 4: 执行布局 ========
        win.setDisplayLayoutNeeded();
        win.mGivenContentInsets.setEmpty();
        win.mGivenVisibleInsets.setEmpty();
        
        // 全局布局
        mWindowPlacerLocked.performSurfacePlacement(true /* force */);
        
        // ======== 阶段 5: 创建/更新 Surface ========
        result = createSurfaceControl(outSurfaceControl, result, win, winAnimator);
        
        // ======== 阶段 6: 输出帧信息 ========
        win.fillClientWindowFramesAndConfiguration(outFrames, mergedConfiguration,
                win.isDrawn() && win.mAttrs.type != TYPE_APPLICATION_STARTING,
                shouldRelayout);
        
        // ======== 阶段 7: 更新输入窗口 ========
        mInputMonitor.updateInputWindowsLw(false /*force*/);
        
        return result;
    }
}
```

### 3.2 布局计算核心

```java
// DisplayContent.java
void performLayout(boolean initial, boolean updateInputWindows) {
    if (!isLayoutNeeded()) {
        return;
    }
    clearLayoutNeeded();
    
    // 1. 开始布局
    final int dw = mDisplayInfo.logicalWidth;
    final int dh = mDisplayInfo.logicalHeight;
    mDisplayPolicy.beginLayoutLw(mDisplayInfo, mDisplayRotation.getRotation());
    
    // 2. 遍历所有窗口计算布局
    forAllWindows(mPerformLayout, true /* traverseTopToBottom */);
    
    // 3. 结束布局
    mDisplayPolicy.finishLayoutLw();
    
    // 4. 更新输入窗口
    if (updateInputWindows) {
        mInputMonitor.updateInputWindowsLw(false /* force */);
    }
}

// 布局单个窗口
private final Consumer<WindowState> mPerformLayout = w -> {
    if (w.mLayoutAttached) {
        return;  // 附加窗口后续处理
    }
    
    if (w.isGoneForLayout()) {
        return;  // 不可见窗口跳过
    }
    
    // 调用 DisplayPolicy 计算窗口帧
    mDisplayPolicy.layoutWindowLw(w, w.getParentWindow());
};
```

### 3.3 DisplayPolicy.layoutWindowLw

```java
// DisplayPolicy.java
public void layoutWindowLw(WindowState win, WindowState attached) {
    // 获取窗口帧
    final WindowFrames windowFrames = win.getWindowFrames();
    
    // 获取显示区域
    final Rect pf = mTmpParentFrame;  // Parent Frame
    final Rect df = mTmpDisplayFrame; // Display Frame
    final Rect cf = mTmpContentFrame; // Content Frame
    final Rect vf = mTmpVisibleFrame; // Visible Frame
    final Rect sf = mTmpStableFrame;  // Stable Frame
    
    final int type = win.mAttrs.type;
    final int fl = win.mAttrs.flags;
    
    // 初始化为全屏
    pf.set(0, 0, mDisplayWidth, mDisplayHeight);
    df.set(pf);
    
    // ======== 根据窗口类型调整 ========
    
    if (type == TYPE_STATUS_BAR || type == TYPE_NAVIGATION_BAR) {
        // 系统 UI 窗口特殊处理
        cf.set(df);
        vf.set(df);
    } else if (attached != null) {
        // 附加窗口使用父窗口的帧
        pf.set(attached.getFrame());
        cf.set(pf);
        vf.set(pf);
    } else {
        // 普通窗口计算可用区域
        cf.set(getContentFrameLw());
        vf.set(getVisibleFrameLw());
    }
    
    // ======== 应用 Insets ========
    if ((fl & FLAG_LAYOUT_IN_SCREEN) == 0) {
        // 不在屏幕内布局，减去系统 Insets
        cf.top += mSystemInsets.top;
        cf.bottom -= mSystemInsets.bottom;
        vf.top += mSystemInsets.top;
        vf.bottom -= mSystemInsets.bottom;
    }
    
    // ======== 计算最终帧 ========
    windowFrames.setFrames(pf, df, cf, vf, sf);
    
    // 考虑 gravity 和请求的大小
    Gravity.apply(win.mAttrs.gravity, win.mRequestedWidth, win.mRequestedHeight,
            cf, win.mAttrs.x, win.mAttrs.y, windowFrames.mFrame);
    
    // 通知窗口帧变化
    win.computeFrame();
}
```

## 4. Surface 创建流程

### 4.1 createSurfaceControl

```java
// WindowManagerService.java
private int createSurfaceControl(SurfaceControl outSurfaceControl, int result,
        WindowState win, WindowStateAnimator winAnimator) {
    
    if (!win.mHasSurface) {
        // 创建 SurfaceControl
        result |= RELAYOUT_RES_SURFACE_CHANGED;
    }
    
    try {
        // 实际创建
        result = winAnimator.createSurfaceLocked(result);
    } catch (OutOfResourcesException e) {
        return 0;
    }
    
    if (result != 0) {
        // 复制到输出参数
        outSurfaceControl.copyFrom(winAnimator.mSurfaceControl, "
                WindowManagerService.createSurfaceControl");
    }
    
    return result;
}
```

### 4.2 WindowStateAnimator.createSurfaceLocked

```java
// WindowStateAnimator.java
int createSurfaceLocked(int result) {
    if (mSurfaceControl == null) {
        // 构建 SurfaceControl
        mSurfaceControl = mWin.makeSurface()
                .setParent(mWin.getSurfaceControl())
                .setName(mWin.mAttrs.getTitle().toString())
                .setFormat(format)
                .setFlags(flags)
                .build();
        
        // 设置初始属性
        mWin.mSession.mSurfaceSession.onSurfaceCreated(mSurfaceControl);
    }
    
    mWin.mHasSurface = true;
    return result;
}
```

## 5. Surface 分配与 BufferQueue

### 5.1 应用获取 Surface

```java
// ViewRootImpl.java
private int relayoutWindow(WindowManager.LayoutParams params, ...) {
    // 调用 WMS relayout
    int relayoutResult = mWindowSession.relayout(
            mWindow, params, ..., mSurfaceControl, ...);
    
    // 从 SurfaceControl 获取 Surface
    if (mSurfaceControl.isValid()) {
        if (!useBLAST()) {
            mSurface.copyFrom(mSurfaceControl);
        } else {
            // BLAST 模式
            updateBlastSurfaceIfNeeded();
        }
    }
    
    return relayoutResult;
}

// 使用 BLAST 时的 Surface 获取
private void updateBlastSurfaceIfNeeded() {
    if (mBlastBufferQueue == null) {
        // 创建 BLASTBufferQueue
        mBlastBufferQueue = new BLASTBufferQueue(mTag, mSurfaceControl,
                mWidth, mHeight, mWindowAttributes.format);
    }
    // 从 BLASTBufferQueue 获取 Surface
    mSurface.transferFrom(mBlastBufferQueue.getSurface());
}
```

## 6. 窗口更新流程

### 6.1 updateViewLayout

```java
// WindowManagerGlobal.java
public void updateViewLayout(View view, ViewGroup.LayoutParams params) {
    final WindowManager.LayoutParams wparams = (WindowManager.LayoutParams) params;
    
    view.setLayoutParams(wparams);
    
    synchronized (mLock) {
        int index = findViewLocked(view, true);
        ViewRootImpl root = mRoots.get(index);
        mParams.remove(index);
        mParams.add(index, wparams);
        root.setLayoutParams(wparams, false);
    }
}

// ViewRootImpl.java
void setLayoutParams(WindowManager.LayoutParams attrs, boolean newView) {
    synchronized (this) {
        mWindowAttributes.copyFrom(attrs);
        // 标记需要重新布局
        scheduleTraversals();
    }
}
```

### 6.2 窗口可见性变化

```java
// ViewRootImpl.java
void handleAppVisibility(boolean visible) {
    if (mAppVisible != visible) {
        mAppVisible = visible;
        mAppVisibilityChanged = true;
        
        // 触发遍历
        scheduleTraversals();
    }
}

// performTraversals 中处理
private void performTraversals() {
    boolean viewVisibilityChanged = mViewVisibility != mAttachInfo.mWindowVisibility
            || mNewSurfaceNeeded || mAppVisibilityChanged;
    
    if (viewVisibilityChanged) {
        // 通知 WMS 可见性变化
        relayoutWindow(params, viewVisibility, ...);
    }
}
```

## 7. 本章小结

本文详细介绍了窗口添加与布局流程：

1. **addWindow 流程**：权限检查 → 创建 WindowState → 创建 SurfaceControl → 更新焦点
2. **relayoutWindow**：更新大小 → 执行布局 → 创建 Surface → 输出帧信息
3. **布局计算**：DisplayPolicy 计算各种帧（Parent/Display/Content/Visible）
4. **Surface 创建**：SurfaceControl.Builder → Native 创建 → SurfaceFlinger Layer
5. **窗口更新**：updateViewLayout 触发 scheduleTraversals

下一篇文章将介绍窗口焦点与输入管理。
