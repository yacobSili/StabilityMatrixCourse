# 文件系统系列文章（共 20 篇）

## 系列概述

本系列从架构师视角全面介绍 Linux/Android 文件系统，涵盖 Framework 层和 Kernel 层，详细阐述文件系统的上下游模块关系、核心机制和实际应用。

### 设计原则

1. **全局架构视角**：第一篇建立 Framework → Kernel → 存储设备 的完整视图
2. **模块关系清晰**：明确上下游、平级交互、无直接联系的模块
3. **中等深度**：聚焦架构设计、关键机制、模块交互，不过度深入实现细节
4. **整合 VFS**：将原 vfs 目录的内容整合并扩展
