# 内存管理系列文章（共 21 篇）

## 系列概述

本系列从架构师视角全面介绍 Linux/Android 内存管理，涵盖应用层、Framework 层和 Kernel 层，详细阐述内存系统的上下游模块关系、核心机制和 Android 特有实现。

### 设计原则

1. **全局架构视角**：第一篇建立 应用层 → Framework → Kernel → 硬件 的完整视图
2. **模块关系清晰**：明确上下游、平级交互、无直接联系的模块
3. **中等深度**：聚焦架构设计、关键机制、模块交互，不过度深入实现细节
4. **Android 特化**：专门章节介绍 Android 内存模型、LMKD、ART
