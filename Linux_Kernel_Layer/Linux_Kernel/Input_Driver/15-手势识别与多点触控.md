# 手势识别与多点触控

## 引言

Android 提供了丰富的手势识别 API，包括 GestureDetector、ScaleGestureDetector 等。本文将详细介绍手势识别机制和多点触控的处理方法。

---

## 1. 手势识别概览

### 1.1 手势类型

```
┌─────────────────────────────────────────────────────────────────────────┐
│                            单点手势                                      │
├─────────────────────────────────────────────────────────────────────────┤
│  点击 (Tap)         │  单击、双击                                       │
│  长按 (Long Press)   │  按住超过阈值时间                                │
│  滑动 (Scroll)       │  按住后移动                                      │
│  快速滑动 (Fling)    │  快速滑动后抬起                                  │
│  按下 (Show Press)   │  按下后短暂延迟                                  │
└─────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────┐
│                            多点手势                                      │
├─────────────────────────────────────────────────────────────────────────┤
│  缩放 (Scale/Pinch)  │  两指距离变化                                    │
│  旋转 (Rotate)       │  两指角度变化                                    │
│  双指滚动            │  两指同时移动                                    │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 手势检测器

| 类 | 用途 |
|---|------|
| GestureDetector | 单点手势 (点击、长按、滑动、Fling) |
| ScaleGestureDetector | 缩放手势 |
| RotationGestureDetector | 旋转手势 (需自定义) |
| VelocityTracker | 速度追踪 |

## 2. GestureDetector

### 2.1 基本使用

```java
public class MyView extends View {
    private GestureDetector mGestureDetector;
    
    public MyView(Context context) {
        super(context);
        
        // 创建 GestureDetector
        mGestureDetector = new GestureDetector(context, 
            new GestureDetector.SimpleOnGestureListener() {
                
                @Override
                public boolean onDown(MotionEvent e) {
                    // 按下 - 必须返回 true 才能接收后续事件
                    return true;
                }
                
                @Override
                public boolean onSingleTapUp(MotionEvent e) {
                    // 单击抬起 (未判定为双击)
                    return true;
                }
                
                @Override
                public boolean onSingleTapConfirmed(MotionEvent e) {
                    // 确认单击 (已排除双击可能)
                    handleSingleTap();
                    return true;
                }
                
                @Override
                public boolean onDoubleTap(MotionEvent e) {
                    // 双击
                    handleDoubleTap();
                    return true;
                }
                
                @Override
                public void onLongPress(MotionEvent e) {
                    // 长按
                    handleLongPress();
                }
                
                @Override
                public boolean onScroll(MotionEvent e1, MotionEvent e2,
                                        float distanceX, float distanceY) {
                    // 滚动
                    scrollBy((int) distanceX, (int) distanceY);
                    return true;
                }
                
                @Override
                public boolean onFling(MotionEvent e1, MotionEvent e2,
                                       float velocityX, float velocityY) {
                    // 快速滑动
                    fling((int) velocityX, (int) velocityY);
                    return true;
                }
                
                @Override
                public void onShowPress(MotionEvent e) {
                    // 按下后短暂延迟 (可用于显示按下状态)
                    showPressedState();
                }
            });
    }
    
    @Override
    public boolean onTouchEvent(MotionEvent event) {
        return mGestureDetector.onTouchEvent(event);
    }
}
```

### 2.2 GestureDetector 内部实现

```java
// frameworks/base/core/java/android/view/GestureDetector.java

public class GestureDetector {
    // 配置常量
    private static final int TAP_TIMEOUT = ViewConfiguration.getTapTimeout();
    private static final int DOUBLE_TAP_TIMEOUT = ViewConfiguration.getDoubleTapTimeout();
    private static final int LONG_PRESS_TIMEOUT = ViewConfiguration.getLongPressTimeout();
    
    // 状态
    private boolean mStillDown;
    private boolean mAlwaysInTapRegion;
    private boolean mAlwaysInBiggerTapRegion;
    private boolean mIsDoubleTapping;
    private boolean mIsLongpressEnabled = true;
    
    // 记录的事件
    private MotionEvent mCurrentDownEvent;
    private MotionEvent mPreviousUpEvent;
    
    // Handler 用于延迟检测
    private final Handler mHandler;
    
    private static final int SHOW_PRESS = 1;
    private static final int LONG_PRESS = 2;
    private static final int TAP = 3;
    
    public boolean onTouchEvent(MotionEvent ev) {
        final int action = ev.getAction();
        
        // 速度追踪
        if (mVelocityTracker == null) {
            mVelocityTracker = VelocityTracker.obtain();
        }
        mVelocityTracker.addMovement(ev);
        
        boolean handled = false;
        
        switch (action & MotionEvent.ACTION_MASK) {
            case MotionEvent.ACTION_DOWN:
                // 检查是否是双击
                if (mPreviousUpEvent != null && isConsideredDoubleTap(
                        mCurrentDownEvent, mPreviousUpEvent, ev)) {
                    mIsDoubleTapping = true;
                    handled |= mDoubleTapListener.onDoubleTap(mCurrentDownEvent);
                    handled |= mDoubleTapListener.onDoubleTapEvent(ev);
                }
                
                // 记录按下事件
                mCurrentDownEvent = MotionEvent.obtain(ev);
                mAlwaysInTapRegion = true;
                mStillDown = true;
                
                // 发送 ShowPress 延迟消息
                mHandler.sendEmptyMessageDelayed(SHOW_PRESS, TAP_TIMEOUT);
                
                // 发送 LongPress 延迟消息
                mHandler.sendEmptyMessageDelayed(LONG_PRESS, 
                        TAP_TIMEOUT + LONG_PRESS_TIMEOUT);
                
                handled |= mListener.onDown(ev);
                break;
                
            case MotionEvent.ACTION_MOVE:
                if (mAlwaysInTapRegion) {
                    // 检查是否移出点击区域
                    final int deltaX = (int) (ev.getX() - mCurrentDownEvent.getX());
                    final int deltaY = (int) (ev.getY() - mCurrentDownEvent.getY());
                    int distance = (deltaX * deltaX) + (deltaY * deltaY);
                    
                    if (distance > mTouchSlopSquare) {
                        // 移出点击区域, 开始滚动
                        mAlwaysInTapRegion = false;
                        mHandler.removeMessages(TAP);
                        mHandler.removeMessages(SHOW_PRESS);
                        mHandler.removeMessages(LONG_PRESS);
                    }
                }
                
                if (!mAlwaysInTapRegion) {
                    // 滚动
                    final float scrollX = mLastFocusX - focusX;
                    final float scrollY = mLastFocusY - focusY;
                    handled = mListener.onScroll(mCurrentDownEvent, ev, 
                                                  scrollX, scrollY);
                    mLastFocusX = focusX;
                    mLastFocusY = focusY;
                }
                break;
                
            case MotionEvent.ACTION_UP:
                mStillDown = false;
                
                // 计算速度
                mVelocityTracker.computeCurrentVelocity(1000, mMaximumFlingVelocity);
                final float velocityX = mVelocityTracker.getXVelocity();
                final float velocityY = mVelocityTracker.getYVelocity();
                
                if (mIsDoubleTapping) {
                    handled |= mDoubleTapListener.onDoubleTapEvent(ev);
                } else if (mAlwaysInTapRegion) {
                    // 在点击区域内抬起
                    handled = mListener.onSingleTapUp(ev);
                    
                    // 发送 TAP 确认 (用于排除双击)
                    mHandler.sendEmptyMessageDelayed(TAP, DOUBLE_TAP_TIMEOUT);
                } else if (isVelocityEnough(velocityX, velocityY)) {
                    // 速度足够, 触发 Fling
                    handled = mListener.onFling(mCurrentDownEvent, ev, 
                                                velocityX, velocityY);
                }
                
                // 记录抬起事件 (用于双击检测)
                mPreviousUpEvent = MotionEvent.obtain(ev);
                
                // 清理
                mVelocityTracker.recycle();
                mVelocityTracker = null;
                mIsDoubleTapping = false;
                
                mHandler.removeMessages(SHOW_PRESS);
                mHandler.removeMessages(LONG_PRESS);
                break;
                
            case MotionEvent.ACTION_CANCEL:
                cancel();
                break;
        }
        
        return handled;
    }
    
    private boolean isConsideredDoubleTap(MotionEvent firstDown,
                                          MotionEvent firstUp,
                                          MotionEvent secondDown) {
        // 检查时间间隔
        long deltaTime = secondDown.getEventTime() - firstUp.getEventTime();
        if (deltaTime > DOUBLE_TAP_TIMEOUT || deltaTime < 0) {
            return false;
        }
        
        // 检查距离
        int deltaX = (int) firstDown.getX() - (int) secondDown.getX();
        int deltaY = (int) firstDown.getY() - (int) secondDown.getY();
        return (deltaX * deltaX + deltaY * deltaY < mDoubleTapSlopSquare);
    }
}
```

## 3. ScaleGestureDetector

### 3.1 基本使用

```java
public class ZoomableImageView extends ImageView {
    private ScaleGestureDetector mScaleDetector;
    private float mScaleFactor = 1.0f;
    
    public ZoomableImageView(Context context) {
        super(context);
        
        mScaleDetector = new ScaleGestureDetector(context,
            new ScaleGestureDetector.SimpleOnScaleGestureListener() {
                
                @Override
                public boolean onScaleBegin(ScaleGestureDetector detector) {
                    // 缩放开始
                    return true;  // 返回 true 表示开始处理
                }
                
                @Override
                public boolean onScale(ScaleGestureDetector detector) {
                    // 缩放中
                    mScaleFactor *= detector.getScaleFactor();
                    
                    // 限制缩放范围
                    mScaleFactor = Math.max(0.1f, Math.min(mScaleFactor, 10.0f));
                    
                    // 获取焦点 (两指中心)
                    float focusX = detector.getFocusX();
                    float focusY = detector.getFocusY();
                    
                    // 应用变换
                    invalidate();
                    return true;
                }
                
                @Override
                public void onScaleEnd(ScaleGestureDetector detector) {
                    // 缩放结束
                }
            });
    }
    
    @Override
    public boolean onTouchEvent(MotionEvent event) {
        mScaleDetector.onTouchEvent(event);
        return true;
    }
    
    @Override
    protected void onDraw(Canvas canvas) {
        canvas.save();
        canvas.scale(mScaleFactor, mScaleFactor);
        super.onDraw(canvas);
        canvas.restore();
    }
}
```

### 3.2 ScaleGestureDetector 内部实现

```java
// frameworks/base/core/java/android/view/ScaleGestureDetector.java

public class ScaleGestureDetector {
    // 状态
    private boolean mInProgress;
    private float mPrevSpan;        // 之前的两指距离
    private float mCurrSpan;        // 当前的两指距离
    private float mFocusX, mFocusY; // 焦点 (两指中心)
    
    public boolean onTouchEvent(MotionEvent event) {
        final int action = event.getActionMasked();
        
        switch (action) {
            case MotionEvent.ACTION_DOWN:
                // 单指按下
                mFocusX = event.getX();
                mFocusY = event.getY();
                break;
                
            case MotionEvent.ACTION_POINTER_DOWN:
                // 第二个手指按下
                if (event.getPointerCount() >= 2) {
                    // 计算初始跨度和焦点
                    calculateSpanAndFocus(event);
                    mPrevSpan = mCurrSpan;
                    mInProgress = mListener.onScaleBegin(this);
                }
                break;
                
            case MotionEvent.ACTION_MOVE:
                if (mInProgress && event.getPointerCount() >= 2) {
                    // 计算当前跨度和焦点
                    calculateSpanAndFocus(event);
                    
                    // 通知监听器
                    boolean updatePrev = mListener.onScale(this);
                    
                    if (updatePrev) {
                        mPrevSpan = mCurrSpan;
                    }
                }
                break;
                
            case MotionEvent.ACTION_POINTER_UP:
                if (mInProgress) {
                    // 只剩一个手指
                    mListener.onScaleEnd(this);
                    mInProgress = false;
                }
                break;
                
            case MotionEvent.ACTION_CANCEL:
            case MotionEvent.ACTION_UP:
                if (mInProgress) {
                    mListener.onScaleEnd(this);
                    mInProgress = false;
                }
                break;
        }
        
        return true;
    }
    
    private void calculateSpanAndFocus(MotionEvent event) {
        final float x0 = event.getX(0);
        final float y0 = event.getY(0);
        final float x1 = event.getX(1);
        final float y1 = event.getY(1);
        
        // 计算焦点 (两指中心)
        mFocusX = (x0 + x1) / 2;
        mFocusY = (y0 + y1) / 2;
        
        // 计算跨度 (两指距离)
        final float dx = x1 - x0;
        final float dy = y1 - y0;
        mCurrSpan = (float) Math.sqrt(dx * dx + dy * dy);
    }
    
    public float getScaleFactor() {
        return mPrevSpan > 0 ? mCurrSpan / mPrevSpan : 1;
    }
    
    public float getFocusX() {
        return mFocusX;
    }
    
    public float getFocusY() {
        return mFocusY;
    }
}
```

### 4.2 多点触控追踪

```java
public class MultiTouchView extends View {
    // 使用 SparseArray 追踪所有触摸点
    private SparseArray<PointF> mActivePointers = new SparseArray<>();
    
    @Override
    public boolean onTouchEvent(MotionEvent event) {
        int action = event.getActionMasked();
        int actionIndex = event.getActionIndex();
        int pointerId;
        
        switch (action) {
            case MotionEvent.ACTION_DOWN:
            case MotionEvent.ACTION_POINTER_DOWN:
                // 新指针按下
                pointerId = event.getPointerId(actionIndex);
                PointF point = new PointF(
                    event.getX(actionIndex), 
                    event.getY(actionIndex));
                mActivePointers.put(pointerId, point);
                break;
                
            case MotionEvent.ACTION_MOVE:
                // 更新所有指针位置
                for (int i = 0; i < event.getPointerCount(); i++) {
                    pointerId = event.getPointerId(i);
                    PointF p = mActivePointers.get(pointerId);
                    if (p != null) {
                        p.x = event.getX(i);
                        p.y = event.getY(i);
                    }
                }
                break;
                
            case MotionEvent.ACTION_UP:
            case MotionEvent.ACTION_POINTER_UP:
                // 指针抬起
                pointerId = event.getPointerId(actionIndex);
                mActivePointers.remove(pointerId);
                break;
                
            case MotionEvent.ACTION_CANCEL:
                // 取消, 清空所有
                mActivePointers.clear();
                break;
        }
        
        invalidate();
        return true;
    }
    
    @Override
    protected void onDraw(Canvas canvas) {
        super.onDraw(canvas);
        
        // 绘制所有触摸点
        for (int i = 0; i < mActivePointers.size(); i++) {
            int pointerId = mActivePointers.keyAt(i);
            PointF point = mActivePointers.valueAt(i);
            
            canvas.drawCircle(point.x, point.y, 50, mPaint);
            canvas.drawText("ID: " + pointerId, point.x, point.y - 60, mTextPaint);
        }
    }
}
```

### 4.3 双指旋转手势

```java
public class RotationGestureDetector {
    private OnRotationGestureListener mListener;
    
    private float mPrevAngle;
    private boolean mInProgress;
    
    public interface OnRotationGestureListener {
        boolean onRotateBegin(RotationGestureDetector detector);
        boolean onRotate(RotationGestureDetector detector);
        void onRotateEnd(RotationGestureDetector detector);
    }
    
    private float mRotation;
    
    public boolean onTouchEvent(MotionEvent event) {
        int action = event.getActionMasked();
        
        switch (action) {
            case MotionEvent.ACTION_POINTER_DOWN:
                if (event.getPointerCount() == 2) {
                    mPrevAngle = calculateAngle(event);
                    mInProgress = mListener.onRotateBegin(this);
                }
                break;
                
            case MotionEvent.ACTION_MOVE:
                if (mInProgress && event.getPointerCount() == 2) {
                    float angle = calculateAngle(event);
                    mRotation = angle - mPrevAngle;
                    
                    boolean updatePrev = mListener.onRotate(this);
                    if (updatePrev) {
                        mPrevAngle = angle;
                    }
                }
                break;
                
            case MotionEvent.ACTION_POINTER_UP:
            case MotionEvent.ACTION_UP:
            case MotionEvent.ACTION_CANCEL:
                if (mInProgress) {
                    mListener.onRotateEnd(this);
                    mInProgress = false;
                }
                break;
        }
        
        return true;
    }
    
    private float calculateAngle(MotionEvent event) {
        float dx = event.getX(1) - event.getX(0);
        float dy = event.getY(1) - event.getY(0);
        return (float) Math.toDegrees(Math.atan2(dy, dx));
    }
    
    public float getRotation() {
        return mRotation;
    }
}
```

## 5. VelocityTracker

### 5.1 基本使用

```java
public class FlingView extends View {
    private VelocityTracker mVelocityTracker;
    
    @Override
    public boolean onTouchEvent(MotionEvent event) {
        int action = event.getActionMasked();
        
        switch (action) {
            case MotionEvent.ACTION_DOWN:
                // 获取 VelocityTracker
                if (mVelocityTracker == null) {
                    mVelocityTracker = VelocityTracker.obtain();
                } else {
                    mVelocityTracker.clear();
                }
                mVelocityTracker.addMovement(event);
                break;
                
            case MotionEvent.ACTION_MOVE:
                mVelocityTracker.addMovement(event);
                break;
                
            case MotionEvent.ACTION_UP:
                // 计算速度
                mVelocityTracker.addMovement(event);
                mVelocityTracker.computeCurrentVelocity(1000,  // 单位: 像素/秒
                        ViewConfiguration.get(getContext()).getScaledMaximumFlingVelocity());
                
                float velocityX = mVelocityTracker.getXVelocity();
                float velocityY = mVelocityTracker.getYVelocity();
                
                // 处理 Fling
                if (Math.abs(velocityX) > mMinimumFlingVelocity ||
                        Math.abs(velocityY) > mMinimumFlingVelocity) {
                    fling((int) velocityX, (int) velocityY);
                }
                
                // 回收
                mVelocityTracker.recycle();
                mVelocityTracker = null;
                break;
                
            case MotionEvent.ACTION_CANCEL:
                if (mVelocityTracker != null) {
                    mVelocityTracker.recycle();
                    mVelocityTracker = null;
                }
                break;
        }
        
        return true;
    }
}
```

## 6. 综合示例: 可缩放拖动的图片

```java
public class ZoomableDraggableImageView extends ImageView {
    private GestureDetector mGestureDetector;
    private ScaleGestureDetector mScaleDetector;
    
    private float mTranslateX = 0;
    private float mTranslateY = 0;
    private float mScaleFactor = 1.0f;
    
    private boolean mIsScaling = false;
    
    public ZoomableDraggableImageView(Context context) {
        super(context);
        init(context);
    }
    
    private void init(Context context) {
        // 手势检测器
        mGestureDetector = new GestureDetector(context,
            new GestureDetector.SimpleOnGestureListener() {
                @Override
                public boolean onScroll(MotionEvent e1, MotionEvent e2,
                                        float distanceX, float distanceY) {
                    if (!mIsScaling) {
                        mTranslateX -= distanceX;
                        mTranslateY -= distanceY;
                        invalidate();
                    }
                    return true;
                }
                
                @Override
                public boolean onDoubleTap(MotionEvent e) {
                    // 双击重置
                    mTranslateX = 0;
                    mTranslateY = 0;
                    mScaleFactor = 1.0f;
                    invalidate();
                    return true;
                }
            });
        
        // 缩放检测器
        mScaleDetector = new ScaleGestureDetector(context,
            new ScaleGestureDetector.SimpleOnScaleGestureListener() {
                @Override
                public boolean onScaleBegin(ScaleGestureDetector detector) {
                    mIsScaling = true;
                    return true;
                }
                
                @Override
                public boolean onScale(ScaleGestureDetector detector) {
                    mScaleFactor *= detector.getScaleFactor();
                    mScaleFactor = Math.max(0.5f, Math.min(mScaleFactor, 5.0f));
                    invalidate();
                    return true;
                }
                
                @Override
                public void onScaleEnd(ScaleGestureDetector detector) {
                    mIsScaling = false;
                }
            });
    }
    
    @Override
    public boolean onTouchEvent(MotionEvent event) {
        mScaleDetector.onTouchEvent(event);
        mGestureDetector.onTouchEvent(event);
        return true;
    }
    
    @Override
    protected void onDraw(Canvas canvas) {
        canvas.save();
        canvas.translate(mTranslateX, mTranslateY);
        canvas.scale(mScaleFactor, mScaleFactor, getWidth() / 2f, getHeight() / 2f);
        super.onDraw(canvas);
        canvas.restore();
    }
}
```

## 7. 本章小结

手势识别与多点触控的要点：

| 组件 | 用途 |
|-----|------|
| GestureDetector | 单点手势: 点击、长按、滑动、Fling |
| ScaleGestureDetector | 缩放手势 |
| VelocityTracker | 速度追踪 |
| MotionEvent 指针 API | 多点触控数据访问 |

关键点：
1. GestureDetector.onDown() 必须返回 true
2. 使用 pointerId 追踪触摸点, 而非 pointerIndex
3. ScaleGestureDetector 的焦点用于确定缩放中心
4. Fling 速度计算需要 VelocityTracker
