# Android Input 系列文章（共 19 篇）

## 系列概述

本系列从架构师视角全面介绍 Android Input 子系统，涵盖 Kernel 层、Native 层、Framework 层和应用层，详细阐述输入事件从硬件到应用的完整流程、焦点管理机制、ANR 处理以及性能调优。

### 设计原则

1. **全局架构视角**：第一篇建立 Kernel → Native → Framework → App 的完整视图
2. **模块关系清晰**：明确上下游、平级交互的模块关系
3. **事件流程导向**：从事件产生到消费的完整链路
4. **实战性强**：包含 ANR 分析、调试工具、性能优化
