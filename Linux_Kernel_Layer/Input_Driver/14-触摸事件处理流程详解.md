# 触摸事件处理流程详解

## 引言

触摸事件 (MotionEvent) 是 Android 设备上最常见的输入事件，其处理流程涉及坐标命中测试、View 层级分发、事件拦截等复杂机制。本文将详细分析触摸事件从 InputDispatcher 到 View 的完整处理流程。

---

## 1. 触摸事件分发概览

### 1.1 分发流程

```
InputDispatcher
    │
    │ dispatchMotionLocked()
    │ findTouchedWindowTargetsLocked()  ← 坐标命中测试
    │
    ▼
┌─────────────────────────────────────────────────────────────────────────┐
│                          InputChannel                                    │
└─────────────────────────────────────────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────────────────────────────────────────┐
│                           应用进程                                       │
│  ViewRootImpl.WindowInputEventReceiver                                  │
│      │                                                                  │
│      ▼                                                                  │
│  ViewPostImeInputStage.processPointerEvent()                           │
│      │                                                                  │
│      ▼                                                                  │
│  DecorView.dispatchTouchEvent()                                        │
│      │                                                                  │
│      ▼                                                                  │
│  Activity.dispatchTouchEvent()                                         │
│      │                                                                  │
│      ▼                                                                  │
│  PhoneWindow.superDispatchTouchEvent()                                 │
│      │                                                                  │
│      ▼                                                                  │
│  ViewGroup.dispatchTouchEvent() ← 分发循环                             │
│      │                                                                  │
│      ├── onInterceptTouchEvent() ← 拦截判断                           │
│      │                                                                  │
│      ├── child.dispatchTouchEvent() ← 分发给子 View                   │
│      │                                                                  │
│      └── onTouchEvent() ← 自身处理                                     │
└─────────────────────────────────────────────────────────────────────────┘
```

### 1.2 MotionEvent Action 详解

```
触摸手势事件序列:

单点触摸:
  DOWN → MOVE → MOVE → ... → UP
  
多点触摸:
  DOWN(pointer0) → POINTER_DOWN(pointer1) → MOVE → ... 
    → POINTER_UP(pointer1) → UP(pointer0)

事件取消:
  DOWN → MOVE → CANCEL

┌─────────────────────────────────────────────────────────────────────────┐
│  Action                 │ 描述                                          │
├─────────────────────────┼─────────────────────────────────────────────────┤
│  ACTION_DOWN           │ 第一个触摸点按下                                │
│  ACTION_UP             │ 最后一个触摸点抬起                              │
│  ACTION_MOVE           │ 触摸点移动                                      │
│  ACTION_CANCEL         │ 事件被取消 (被父 View 拦截)                     │
│  ACTION_POINTER_DOWN   │ 非第一个触摸点按下 (多点触控)                   │
│  ACTION_POINTER_UP     │ 非最后一个触摸点抬起 (多点触控)                 │
│  ACTION_OUTSIDE        │ 触摸发生在窗口外                                │
└─────────────────────────┴─────────────────────────────────────────────────┘
```

## 2. InputDispatcher 触摸分发

### 2.1 dispatchMotionLocked

```cpp
// frameworks/native/services/inputflinger/dispatcher/InputDispatcher.cpp

bool InputDispatcher::dispatchMotionLocked(nsecs_t currentTime,
                                            std::shared_ptr<MotionEntry> entry,
                                            DropReason* dropReason,
                                            nsecs_t* nextWakeupTime) {
    // 检查是否是指针事件 (触摸/鼠标)
    bool isPointerEvent = entry->source & AINPUT_SOURCE_CLASS_POINTER;
    
    std::vector<InputTarget> inputTargets;
    InputEventInjectionResult injectionResult;
    
    if (isPointerEvent) {
        // 触摸事件 - 根据坐标查找目标窗口
        injectionResult = findTouchedWindowTargetsLocked(
            currentTime, *entry, inputTargets, nextWakeupTime);
    } else {
        // 非指针事件 - 发送到焦点窗口
        injectionResult = findFocusedWindowTargetsLocked(
            currentTime, *entry, inputTargets, nextWakeupTime);
    }
    
    if (injectionResult == InputEventInjectionResult::PENDING) {
        return false;  // 等待
    }
    
    if (injectionResult != InputEventInjectionResult::SUCCEEDED) {
        *dropReason = DropReason::NO_POINTER_CAPTURE;
        return true;
    }
    
    // 分发到目标窗口
    dispatchEventLocked(currentTime, entry, inputTargets);
    return true;
}
```

### 2.2 findTouchedWindowTargetsLocked

```cpp
InputEventInjectionResult InputDispatcher::findTouchedWindowTargetsLocked(
        nsecs_t currentTime,
        const MotionEntry& entry,
        std::vector<InputTarget>& inputTargets,
        nsecs_t* nextWakeupTime) {
    
    int32_t displayId = entry.displayId;
    int32_t action = entry.action;
    int32_t maskedAction = action & AMOTION_EVENT_ACTION_MASK;
    
    // 获取或创建触摸状态
    TouchState& state = mTouchStatesByDisplay[displayId];
    
    // DOWN 事件 - 查找命中的窗口
    if (maskedAction == AMOTION_EVENT_ACTION_DOWN) {
        // 重置触摸状态
        state.reset();
        
        // 获取触摸坐标
        float x = entry.pointerCoords[0].getAxisValue(AMOTION_EVENT_AXIS_X);
        float y = entry.pointerCoords[0].getAxisValue(AMOTION_EVENT_AXIS_Y);
        
        // 查找命中的窗口
        sp<WindowInfoHandle> touchedWindow = 
            findTouchedWindowAtLocked(displayId, x, y);
        
        if (touchedWindow != nullptr) {
            // 检查窗口是否可以接收输入
            const WindowInfo& info = *touchedWindow->getInfo();
            
            // 检查是否被遮挡
            if (isWindowObscuredLocked(touchedWindow, x, y)) {
                // 被遮挡, 可能需要拒绝
                if (info.inputFeatures.test(
                        WindowInfo::Feature::DROP_INPUT_IF_OBSCURED)) {
                    return InputEventInjectionResult::FAILED;
                }
            }
            
            // 添加到触摸状态
            state.addOrUpdateWindow(touchedWindow, 
                                    InputTarget::FLAG_FOREGROUND |
                                    InputTarget::FLAG_DISPATCH_AS_IS,
                                    BitSet32(0).markBit(0));
        }
    }
    
    // POINTER_DOWN 事件 - 新的触摸点
    else if (maskedAction == AMOTION_EVENT_ACTION_POINTER_DOWN) {
        int32_t pointerIndex = getMotionEventActionPointerIndex(action);
        float x = entry.pointerCoords[pointerIndex].getAxisValue(AMOTION_EVENT_AXIS_X);
        float y = entry.pointerCoords[pointerIndex].getAxisValue(AMOTION_EVENT_AXIS_Y);
        
        // 查找新触摸点命中的窗口
        sp<WindowInfoHandle> newWindow = findTouchedWindowAtLocked(displayId, x, y);
        
        // 处理分割触摸
        if (newWindow != nullptr && newWindow != state.getFirstForegroundWindow()) {
            // 新触摸点在不同窗口, 需要分割触摸
            state.addOrUpdateWindow(newWindow, 
                                    InputTarget::FLAG_FOREGROUND |
                                    InputTarget::FLAG_DISPATCH_AS_SLIPPERY_ENTER,
                                    BitSet32(0).markBit(pointerIndex));
        }
    }
    
    // MOVE 事件 - 所有触摸点移动
    else if (maskedAction == AMOTION_EVENT_ACTION_MOVE) {
        // 使用已有的触摸状态
    }
    
    // UP 事件 - 最后一个触摸点抬起
    else if (maskedAction == AMOTION_EVENT_ACTION_UP) {
        // 分发后清理触摸状态
    }
    
    // 构建输入目标
    for (const TouchedWindow& touchedWindow : state.windows) {
        InputTarget target;
        target.inputChannel = getInputChannelLocked(
            touchedWindow.windowHandle->getToken());
        target.flags = touchedWindow.targetFlags;
        target.pointerIds = touchedWindow.pointerIds;
        
        // 计算坐标偏移 (窗口坐标系)
        const WindowInfo& info = *touchedWindow.windowHandle->getInfo();
        target.xOffset = -info.frameLeft;
        target.yOffset = -info.frameTop;
        target.globalScaleFactor = info.globalScaleFactor;
        
        inputTargets.push_back(target);
    }
    
    return InputEventInjectionResult::SUCCEEDED;
}
```

### 3.1 ViewRootImpl 接收

```java
// frameworks/base/core/java/android/view/ViewRootImpl.java

final class ViewPostImeInputStage extends InputStage {
    @Override
    protected int onProcess(QueuedInputEvent q) {
        if (q.mEvent instanceof MotionEvent) {
            return processPointerEvent(q);
        }
        return FORWARD;
    }
    
    private int processPointerEvent(QueuedInputEvent q) {
        final MotionEvent event = (MotionEvent) q.mEvent;
        
        // 分发给 View 层次结构
        boolean handled = mView.dispatchPointerEvent(event);
        
        if (handled) {
            return FINISH_HANDLED;
        }
        
        return FORWARD;
    }
}
```

```java
// frameworks/base/core/java/android/app/Activity.java

public boolean dispatchTouchEvent(MotionEvent ev) {
    // 通知用户活动
    if (ev.getAction() == MotionEvent.ACTION_DOWN) {
        onUserInteraction();
    }
    
    // 委托给 Window 分发
    if (getWindow().superDispatchTouchEvent(ev)) {
        return true;
    }
    
    // Window 未处理, Activity 处理
    return onTouchEvent(ev);
}

public boolean onTouchEvent(MotionEvent event) {
    // 检查是否应该关闭 (点击窗口外)
    if (mWindow.shouldCloseOnTouch(this, event)) {
        finish();
        return true;
    }
    
    return false;
}
```

## 4. ViewGroup 事件分发

### 4.1 dispatchTouchEvent 核心实现

```java
// frameworks/base/core/java/android/view/ViewGroup.java

@Override
public boolean dispatchTouchEvent(MotionEvent ev) {
    boolean handled = false;
    
    // 安全过滤
    if (onFilterTouchEventForSecurity(ev)) {
        final int action = ev.getAction();
        final int actionMasked = action & MotionEvent.ACTION_MASK;
        
        // DOWN 事件时重置状态
        if (actionMasked == MotionEvent.ACTION_DOWN) {
            cancelAndClearTouchTargets(ev);
            resetTouchState();
        }
        
        // 检查是否拦截
        final boolean intercepted;
        if (actionMasked == MotionEvent.ACTION_DOWN
                || mFirstTouchTarget != null) {
            // DOWN 事件或已有触摸目标时检查拦截
            final boolean disallowIntercept = 
                (mGroupFlags & FLAG_DISALLOW_INTERCEPT) != 0;
            
            if (!disallowIntercept) {
                intercepted = onInterceptTouchEvent(ev);
            } else {
                intercepted = false;
            }
        } else {
            // 没有触摸目标, 直接拦截
            intercepted = true;
        }
        
        // 如果未拦截, 查找子 View
        if (!intercepted) {
            if (actionMasked == MotionEvent.ACTION_DOWN
                    || (actionMasked == MotionEvent.ACTION_POINTER_DOWN)) {
                
                // 遍历子 View (逆序, 从上到下)
                for (int i = childrenCount - 1; i >= 0; i--) {
                    final View child = getAndVerifyPreorderedView(
                            preorderedList, children, childIndex);
                    
                    // 检查子 View 是否可以接收事件
                    if (!canViewReceivePointerEvents(child)
                            || !isTransformedTouchPointInView(x, y, child, null)) {
                        continue;
                    }
                    
                    // 分发给子 View
                    if (dispatchTransformedTouchEvent(ev, false, child, idBitsToAssign)) {
                        // 子 View 处理了, 记录触摸目标
                        newTouchTarget = addTouchTarget(child, idBitsToAssign);
                        alreadyDispatchedToNewTouchTarget = true;
                        break;
                    }
                }
            }
        }
        
        // 分发给触摸目标
        if (mFirstTouchTarget == null) {
            // 没有触摸目标, 自己处理
            handled = dispatchTransformedTouchEvent(ev, canceled, null,
                    TouchTarget.ALL_POINTER_IDS);
        } else {
            // 分发给所有触摸目标
            TouchTarget predecessor = null;
            TouchTarget target = mFirstTouchTarget;
            
            while (target != null) {
                final TouchTarget next = target.next;
                
                if (alreadyDispatchedToNewTouchTarget && target == newTouchTarget) {
                    handled = true;
                } else {
                    final boolean cancelChild = 
                        intercepted || resetCancelNextUpFlag(target.child);
                    
                    if (dispatchTransformedTouchEvent(ev, cancelChild,
                            target.child, target.pointerIdBits)) {
                        handled = true;
                    }
                }
                
                predecessor = target;
                target = next;
            }
        }
        
        // 清理
        if (actionMasked == MotionEvent.ACTION_UP
                || actionMasked == MotionEvent.ACTION_CANCEL) {
            resetTouchState();
        }
    }
    
    return handled;
}
```

### 4.2 坐标转换

```java
private boolean dispatchTransformedTouchEvent(MotionEvent event,
        boolean cancel, View child, int desiredPointerIdBits) {
    
    final boolean handled;
    
    // 取消事件
    if (cancel || oldAction == MotionEvent.ACTION_CANCEL) {
        event.setAction(MotionEvent.ACTION_CANCEL);
        if (child == null) {
            handled = super.dispatchTouchEvent(event);
        } else {
            handled = child.dispatchTouchEvent(event);
        }
        event.setAction(oldAction);
        return handled;
    }
    
    // 分发给子 View, 需要坐标转换
    if (child == null) {
        handled = super.dispatchTouchEvent(event);
    } else {
        // 计算偏移量
        final float offsetX = mScrollX - child.mLeft;
        final float offsetY = mScrollY - child.mTop;
        
        // 转换坐标
        event.offsetLocation(offsetX, offsetY);
        
        // 分发
        handled = child.dispatchTouchEvent(event);
        
        // 恢复坐标
        event.offsetLocation(-offsetX, -offsetY);
    }
    
    return handled;
}
```

### 4.3 拦截机制

```java
// 父 View 拦截
public boolean onInterceptTouchEvent(MotionEvent ev) {
    // 默认不拦截
    return false;
}

// 子 View 禁止父 View 拦截
public void requestDisallowInterceptTouchEvent(boolean disallowIntercept) {
    if (disallowIntercept != ((mGroupFlags & FLAG_DISALLOW_INTERCEPT) != 0)) {
        if (disallowIntercept) {
            mGroupFlags |= FLAG_DISALLOW_INTERCEPT;
        } else {
            mGroupFlags &= ~FLAG_DISALLOW_INTERCEPT;
        }
        
        // 向上传递
        if (mParent != null) {
            mParent.requestDisallowInterceptTouchEvent(disallowIntercept);
        }
    }
}
```

## 5. View 事件处理

### 5.1 dispatchTouchEvent

```java
// frameworks/base/core/java/android/view/View.java

public boolean dispatchTouchEvent(MotionEvent event) {
    boolean result = false;
    
    // 安全过滤
    if (onFilterTouchEventForSecurity(event)) {
        // 调试
        if ((mViewFlags & ENABLED_MASK) == ENABLED && handleScrollBarDragging(event)) {
            result = true;
        }
        
        // OnTouchListener 优先处理
        ListenerInfo li = mListenerInfo;
        if (li != null && li.mOnTouchListener != null
                && (mViewFlags & ENABLED_MASK) == ENABLED
                && li.mOnTouchListener.onTouch(this, event)) {
            result = true;
        }
        
        // 如果监听器未处理, 调用 onTouchEvent
        if (!result && onTouchEvent(event)) {
            result = true;
        }
    }
    
    return result;
}
```

### 5.2 onTouchEvent

```java
public boolean onTouchEvent(MotionEvent event) {
    final float x = event.getX();
    final float y = event.getY();
    final int action = event.getAction();
    
    // 检查是否可点击
    final boolean clickable = ((mViewFlags & CLICKABLE) == CLICKABLE
            || (mViewFlags & LONG_CLICKABLE) == LONG_CLICKABLE
            || (mViewFlags & CONTEXT_CLICKABLE) == CONTEXT_CLICKABLE);
    
    if (!clickable) {
        // 不可点击的 View 不处理触摸事件
        return false;
    }
    
    switch (action) {
        case MotionEvent.ACTION_DOWN:
            // 开始追踪
            mHasPerformedLongPress = false;
            
            // 设置按下状态
            setPressed(true, x, y);
            
            // 检查长按
            checkForLongClick(0, x, y);
            break;
            
        case MotionEvent.ACTION_MOVE:
            // 检查是否移出视图
            if (!pointInView(x, y, mTouchSlop)) {
                // 移出, 取消按下状态
                removeTapCallback();
                removeLongPressCallback();
                
                if ((mPrivateFlags & PFLAG_PRESSED) != 0) {
                    setPressed(false);
                }
            }
            break;
            
        case MotionEvent.ACTION_UP:
            boolean prepressed = (mPrivateFlags & PFLAG_PREPRESSED) != 0;
            
            if ((mPrivateFlags & PFLAG_PRESSED) != 0 || prepressed) {
                // 在按下状态抬起
                boolean focusTaken = false;
                
                if (isFocusable() && isFocusableInTouchMode() && !isFocused()) {
                    focusTaken = requestFocus();
                }
                
                if (!mHasPerformedLongPress && !mIgnoreNextUpEvent) {
                    // 移除长按检测
                    removeLongPressCallback();
                    
                    // 执行点击
                    if (!focusTaken) {
                        if (mPerformClick == null) {
                            mPerformClick = new PerformClick();
                        }
                        
                        if (!post(mPerformClick)) {
                            performClick();
                        }
                    }
                }
                
                // 清除按下状态
                setPressed(false);
            }
            break;
            
        case MotionEvent.ACTION_CANCEL:
            // 取消
            setPressed(false);
            removeTapCallback();
            removeLongPressCallback();
            break;
    }
    
    return true;
}
```

### 5.3 点击和长按

```java
public boolean performClick() {
    // 发送无障碍事件
    sendAccessibilityEvent(AccessibilityEvent.TYPE_VIEW_CLICKED);
    
    // 调用点击监听器
    ListenerInfo li = mListenerInfo;
    if (li != null && li.mOnClickListener != null) {
        playSoundEffect(SoundEffectConstants.CLICK);
        li.mOnClickListener.onClick(this);
        return true;
    }
    
    return false;
}

private void checkForLongClick(int delayOffset, float x, float y) {
    if ((mViewFlags & LONG_CLICKABLE) == LONG_CLICKABLE
            || (mViewFlags & TOOLTIP) == TOOLTIP) {
        
        mHasPerformedLongPress = false;
        
        // 延迟执行长按检测
        postDelayed(mPendingCheckForLongPress,
                ViewConfiguration.getLongPressTimeout() - delayOffset);
    }
}

private boolean performLongClick(float x, float y) {
    mHasPerformedLongPress = true;
    
    boolean handled = false;
    ListenerInfo li = mListenerInfo;
    
    if (li != null && li.mOnLongClickListener != null) {
        handled = li.mOnLongClickListener.onLongClick(this);
    }
    
    if (!handled) {
        handled = showContextMenu(x, y);
    }
    
    if (handled) {
        performHapticFeedback(HapticFeedbackConstants.LONG_PRESS);
    }
    
    return handled;
}
```

## 6. 事件分发总结

### 6.1 事件分发伪代码

```
ViewGroup.dispatchTouchEvent(event):
    // 1. DOWN 事件时重置状态
    if (action == DOWN):
        clearTouchTargets()
    
    // 2. 检查是否拦截
    if (action == DOWN || hasTouchTarget):
        if (!disallowIntercept):
            intercepted = onInterceptTouchEvent(event)
        else:
            intercepted = false
    else:
        intercepted = true
    
    // 3. 如果未拦截, 查找子 View
    if (!intercepted && action == DOWN):
        for child in children (reversed):
            if (child.contains(x, y)):
                if (child.dispatchTouchEvent(event)):
                    touchTarget = child
                    break
    
    // 4. 分发事件
    if (touchTarget == null):
        handled = onTouchEvent(event)  // 自己处理
    else:
        handled = touchTarget.dispatchTouchEvent(event)
    
    return handled
```

## 7. 本章小结

触摸事件处理的核心机制：

| 机制 | 描述 |
|-----|------|
| 命中测试 | InputDispatcher 根据坐标查找目标窗口 |
| 分发链 | Activity → Window → DecorView → ViewGroup... |
| 拦截机制 | onInterceptTouchEvent / requestDisallowInterceptTouchEvent |
| 触摸目标 | DOWN 时确定, 后续事件直接发送 |
| 坐标转换 | 每层 ViewGroup 转换到子 View 坐标系 |

关键点：
1. DOWN 事件决定触摸目标
2. 父 View 可以拦截子 View 的事件
3. 子 View 可以禁止父 View 拦截
4. OnTouchListener 优先于 onTouchEvent
